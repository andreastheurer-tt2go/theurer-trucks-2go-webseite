3 Strategieansätze für die neue TT2GO Webseite

Alle Ansätze sind One-Pager mit dem Ziel: fleetster-Registrierung maximieren. Der Unterschied liegt in der Erzählstrategie — wie führen wir "Julia" zur Registrierung?

A

Angst-zu-Lösung Funnel

Strategie: Direkt die Ängste der Zielgruppe ansprechen und auflösen. Die Seite erzählt die Geschichte: "Wir verstehen dein Problem → So lösen wir es → Registriere dich"

Seitenstruktur:

  1. Hero: Emotionale Headline ("Dein Pferd verdient einen sicheren Transport") + Video-Loop + CTA
  2. Problem-Sektion: "Kennst du das?" — Ängste direkt benennen (Verladen, kein Führerschein, kaputter Anhänger)
  3. Lösung: "Mit TheurerTrucks 2GO ist das anders" — Vorteile als Antwort auf jede Angst
  4. Video: Vimeo-Erklärvideo
  5. Fahrzeug-Showcase: Features die Sicherheit betonen
  6. So geht's: 4 Schritte
  7. Trustpilot: Echte Bewertungen von Erstnutzerinnen
  8. Standortkarte + Verfügbarkeit
  9. Preise
  10. FAQ
  11. Finaler CTA

Vorteile

  • Spricht Emotionen direkt an — hohe Identifikation
  • Baut systematisch Vertrauen auf
  • Ideal für Facebook-Traffic (emotional)

Nachteile

  • Kann abschreckend wirken, wenn zu stark auf Angst fokussiert
  • Weniger geeignet für Wiederkehr-Nutzer
B

Produkt-First Showcase

Strategie: Das Fahrzeug und die Einfachheit in den Vordergrund stellen. "Schau wie premium unser Transporter ist → So einfach funktioniert's → Registriere dich"

Seitenstruktur:

  1. Hero: Beeindruckende Truck-Visuals + Video-Loop + "Jetzt registrieren"
  2. Video: Vimeo-Erklärvideo prominent oben
  3. Fahrzeug-Showcase: Große Bildergalerie, alle Features im Detail
  4. Verfügbarkeit + Standortkarte
  5. So geht's: 4 Schritte
  6. Preise
  7. Vorteile: Klasse B, Versicherung, 24/7
  8. Trustpilot
  9. FAQ
  10. Finaler CTA

Vorteile

  • Starke visuelle Wirkung
  • Gut für Google-Traffic (Nutzer sucht aktiv)
  • Zeigt Qualität sofort

Nachteile

  • Adressiert Ängste nur indirekt
  • Weniger emotionaler Hook für kalten Traffic
C

Empathie-Hybrid (Empfehlung)

Strategie: Kombiniert emotionalen Einstieg mit starkem Produkt-Showcase. Beginnt empathisch, löst dann über das Fahrzeug und die Einfachheit auf. Mehrere CTAs entlang des Funnels.

Seitenstruktur:

  1. Hero: Empathische Headline + Video-Loop + Statistik-Bar + primärer CTA
  2. Verfügbarkeitsprüfung: Direkt unter Hero — erste Interaktion, Commitment erzeugen
  3. Vertrauens-Sektion: "Entspannt transportieren statt Stress" — 3-4 Schmerzpunkte kurz benennen + wie TT2GO sie löst (keine lange Angst-Sektion, sondern knapp und lösungsorientiert)
  4. Video-Showcase: Stiller Loop + Vimeo-Erklärvideo + zweiter CTA
  5. Fahrzeug: STX im Detail mit Bildergalerie — Features die Sicherheit & Komfort betonen
  6. So geht's: 4 Schritte (Registrieren → Standort wählen → Buchen → Losfahren)
  7. Standortkarte: Interaktiv mit PLZ-Suche + dritter CTA
  8. Trustpilot-Bewertungen: Echte Stimmen
  9. Preise & Tarife: Transparent + vierter CTA
  10. FAQ: Akkordeon mit SEO-relevanten Fragen
  11. Finaler CTA-Band: "Bereit für deinen ersten Transport?"

Vorteile

  • Spricht beide Traffic-Quellen an (Facebook = emotional, Google = intentional)
  • Mehrere CTAs fangen Nutzer an verschiedenen Entscheidungspunkten ab
  • Empathischer Einstieg ohne zu stark auf Angst zu drücken
  • Baut auf bestehendem Testprojekt auf — am wenigsten Umbau nötig

Nachteile

  • Etwas längere Seite als reiner Produkt-Showcase
